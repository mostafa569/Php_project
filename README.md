# Cafeteria Management System  

## 📌 Project Overview  
The **Cafeteria Management System** is a web-based application designed to streamline cafeteria operations by enabling **users** to place and manage their orders while allowing **admins** to handle users, products, and order statuses efficiently.  

---

## 🚀 Features  

### 👤 User Features  
- 🔐 **Login & Reset Password** – Secure authentication using Gmail.  
- 🍔 **Place Orders** – Browse all available products and order easily.  
- 📜 **View Order History** – Track all past orders.  
- ❌ **Cancel Orders** – Cancel orders if they are still in the "Processing" status.  

### 🛠 Admin Features  
- 🏷 **Place Orders for Users** – Admins can manually add orders for specific users.  
- 👥 **Manage Users** – Perform **CRUD operations** (Create, Read, Update, Delete) on users.  
- 📦 **Manage Products** – Add, update, or delete products from the system.  
- 📊 **Manage Orders** – View all orders and update their statuses.  

---

## 🏗️ Tech Stack  
- **Frontend:** HTML, CSS, Bootstrap, JavaScript 
- **Backend:** PHP, MySQL  
- **Authentication:** Gmail-based authentication  
- **Session Management:** PHP Sessions  
- **Database:** MySQL
